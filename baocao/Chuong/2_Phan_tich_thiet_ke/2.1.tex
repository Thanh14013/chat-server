Cơ chế tương tác giữa các phân tầng:

\begin{enumerate}
    \item Tầng Mạng và Cốt lõi \\
% REMOVED BLANK LINE
    Tương tác với Socket API theo mô hình phi đồng bộ để xử lý I/O mạng.
    \begin{itemize}[label={-}]
        \item \texttt{TcpServer}: Lắng nghe kết nối TCP và cấp phát tài nguyên.
        \item \texttt{EventLoop}: Điều phối sự kiện mạng thông qua I/O đa kênh.
        \item \texttt{ThreadPool}: Xử lý các tác vụ CPU-bound.
        \item \texttt{ClientSession}: Quản lý trạng thái phiên, bộ đệm và tham chiếu khóa.
    \end{itemize}

    \item Tầng Bảo mật và Lọc \\
% REMOVED BLANK LINE
    Kiểm duyệt dữ liệu mạng trước khi chuyển giao cho tầng nghiệp vụ.
    \begin{itemize}[label={-}]
        \item \texttt{KeyExchange} \& \texttt{CryptoEngine}: Trao đổi khóa và thực thi mã hóa/giải mã.
        \item \texttt{RateLimiter}: Kiểm soát tần suất yêu cầu.
        \item \texttt{IntrusionDetector} \& \texttt{MessageFilter}: Phát hiện bất thường và làm sạch dữ liệu đầu vào.
        \item \texttt{AuditLogger}: Ghi nhận sự kiện hệ thống.
    \end{itemize}

    \item Tầng Nghiệp vụ \\
% REMOVED BLANK LINE
    Thực thi logic ứng dụng.
    \begin{itemize}[label={-}]
        \item \texttt{AuthManager}: Xác thực định danh và quản lý phiên.
        \item \texttt{RoomManager} \& \texttt{Room}: Quản lý vòng đời phòng và định tuyến thông điệp.
        \item \texttt{FileTransfer}: Phân mảnh và tái tổ hợp dữ liệu tệp.
        \item \texttt{AdminCommands}: Xử lý lệnh quản trị.
    \end{itemize}
\end{enumerate}
